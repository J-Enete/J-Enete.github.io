Nestuge

UX case study: Enhancing User Experience for Nestuge

An onboarding automation platform for communities, courses, and events.

  • Platform: Desktop and Mobile (website/webapp)
  • Client: Nestuge Ltd.
  • Industry: Productivity, e-Commerce
  • Live: Yes (app.nestuge.com)
  • Design team: 3 persons
  • My role: Lead UX designer

Background

Nestuge, a startup co-founded by Nelson Eze, Jude Enete and Ruth Okoli, aimed to revolutionize project management and collaboration. The platform sought to streamline tasks, improve communication, and provide an intuitive user experience. This case study highlights key UX initiatives undertaken to enhance Nestuge's user interface and overall user satisfaction.

Challenge

The initial platform faced usability issues and lacked a cohesive design, hindering user adoption. The challenge was to restructure the interface, making it more intuitive, visually appealing, and conducive to efficient project management.

Approach

  1. User Research: Conducted surveys and interviews with current users to understand pain points and expectations. Identified common tasks and user preferences to inform redesign.
  2. Information Architecture: Restructured the platform's information architecture to ensure logical flow. Categorized features for better accessibility, reducing the cognitive load on users.
  3. Prototyping: Developed low-fidelity wireframes and prototypes to visualize the new design. Solicited feedback from a sample user group to iterate and refine the interface.
  4. Visual Design: Introduced a clean and modern design language. Prioritized a consistent color scheme, intuitive icons, and a visually pleasing layout to improve overall aesthetics.
  5. Task Flows and Navigation: Streamlined task flows to reduce the number of clicks required for common actions. Implemented intuitive navigation, ensuring users could easily find and access essential features.
  6. Collaborative Features: Enhanced collaborative features, allowing team members to share files seamlessly, provide real-time updates, and communicate effortlessly within the platform.

Outcome

The revamped Nestuge platform witnessed a significant improvement in user satisfaction and engagement. Key performance indicators, such as user retention and task completion rates, showed positive trends. Users reported a more intuitive and enjoyable experience, leading to increased productivity and effective collaboration among team members.

Key Takeaways

  1. User-Centric Design: Prioritize user needs and preferences to create a platform that aligns with the user's mental model.
  2. Iterative Prototyping: Continuously gather feedback and iterate on prototypes to refine the design based on real user insights.
  3. Visual Consistency: Maintain a consistent visual design language to create a cohesive and professional user interface.
  4. Efficient Navigation: Simplify task flows and navigation to minimize user effort in achieving their goals.
